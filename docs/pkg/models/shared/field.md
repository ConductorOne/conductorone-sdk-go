# Field

The Field message.

This message contains a oneof named field. Only a single field of the following list may be set at a time:
  - str
  - select
  - random
  - import
  - oauth2
  - readOnly
  - options
  - checkbox
  - secret
  - strList
  - text
  - keyValue
  - stringMap
